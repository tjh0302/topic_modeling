# topic_modeling

## Unsupervised topic modeling using Latent Dirichlet Allocation

The notebook and python module include implementation of LDA for unsupervised topic modeling on a set of text. 

Functions are built out to visualize the model's topics, examine topics, identify the most probabilistic title, and map topics back to a pandas dataframe.

## Example data

Created using ChatGPT my asking it to compile 100 sentences about finance, energy, and education.
